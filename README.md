# DigitalWatermarking
A software that can easily create digital watermarking. 



## Development

1. Install the dependencies first.
2. Solution Platform - x86.
3. Build.

### Dependencies

Windows 10

Visual Studio 2017

- .Net
- C++

  - C++/CLI
